# NODE Backend

Backend API built with Node.js, Express.js, and MongoDB.

## Installation

```bash
git clone https://github.com/idipangkar-ai/node-dev-basics.git
npm install
```

## Environment Variables

Create a `.env` file:

```env
PORT=5000
MONGO_URI=
```

## Run Locally

```bash
npm run dev
```

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
